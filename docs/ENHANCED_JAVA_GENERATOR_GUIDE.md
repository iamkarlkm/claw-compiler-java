# EnhancedJavaCodeGenerator 使用指南

## 概述

EnhancedJavaCodeGenerator 是一个功能完整的 Java 代码生成器，实现了 100% 的 Java 语言支持，包括泛型、错误处理、代码优化等高级特性。

## 主要特性

### 1. 泛型支持
- 完整的泛型类型映射
- 泛型集合自动生成（List<T>, Map<K,V>）
- 泛型工具类生成

### 2. 增强的错误处理
- try-catch-finally 完整支持
- 自定义异常类生成
- 错误收集和报告

### 3. 完整的指令支持
- 所有 IR 指令的正确映射
- 流控制语句生成
- 内存管理代码生成

### 4. 更好的代码组织
- 自动包声明生成
- 智能导入管理
- 类和方法统计

### 5. 完善的注解处理
- 文件头注释生成
- Javadoc 注解转换
- 代码统计信息

## 使用方法

### 基本使用

```java
// 创建生成器
EnhancedJavaCodeGenerator generator = new EnhancedJavaCodeGenerator();

// 创建 IR
ClawIR ir = createClawIR();

// 生成代码
String javaCode = generator.generate(ir);
```

### 使用配置

```java
// 创建配置
GenerationConfig config = new GenerationConfig();
config.setGenerateComments(true);
config.setOutputDirectory("generated");
config.setIncludeHelpers(true);

// 生成代码
GenerationResult result = generator.generate(ir, config);

// 获取生成的文件
for (Map.Entry<String, String> file : result.getFiles().entrySet()) {
    System.out.println(file.getKey() + ": " + file.getValue());
}

// 获取统计信息
System.out.println("统计信息: " + result.getStats());
```

## 最佳实践

1. **使用配置**：总是使用 GenerationConfig 来控制代码生成
2. **检查错误**：始终检查 GenerationResult 中的错误和警告
3. **利用统计信息**：使用统计信息来监控生成的代码质量
4. **启用辅助类**：启用 includeHelpers 来获得更好的泛型支持
5. **添加注释**：启用 generateComments 来生成文档

## 故障排除

### 常见问题

1. **导入缺失**：确保生成器配置中启用了必要的导入
2. **类型不匹配**：检查类型映射是否正确
3. **生成失败**：查看 GenerationResult 中的错误信息

### 调试技巧

1. 启用详细日志
2. 使用生成的代码示例
3. 逐步测试每个功能

EnhancedJavaCodeGenerator 提供了完整的 Java 代码生成功能，支持所有现代 Java 特性，包括泛型、异常处理、流控制等。通过使用配置和检查生成结果，可以生成高质量的、可运行的 Java 代码。
