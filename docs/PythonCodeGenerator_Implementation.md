# PythonCodeGenerator 实现文档

## 概述

PythonCodeGenerator 是 Claw 编译器的 Python 目标代码生成器，负责将 Claw 语言中间表示 (ClawIR) 转换为 Python 3 代码。

## 功能特性

支持生成以下操作码的 Python 代码：

- `LOAD_VAR` - 加载变量
- `STORE_VAR` - 存储变量

### 4. 输出示例

#### 完整的 Python 代码生成示例

```python
#!/usr/bin/env python3
# Auto-generated by Claw Compiler
# Target: Python 3.10+
# Source: test.claw

from __future__ import annotations
from typing import Any, Final, Optional, List, Dict, Callable, Union, overload

# ========================================
# Claw Runtime Helpers for Python
# ========================================

# --- Property Monitor Helpers ---
class ClawPropertyMonitor:
    """属性变更监听基类"""
    def __init__(self):
        self._property_listeners_before: Dict[str, List[Callable]] = {}
        self._property_listeners_after: Dict[str, List[Callable]] = {}

    def _register_before_listener(self, property_path: str, callback: Callable) -> None:
        if property_path not in self._property_listeners_before:
            self._property_listeners_before[property_path] = []
        self._property_listeners_before[property_path].append(callback)

    def _register_after_listener(self, property_path: str, callback: Callable) -> None:
        if property_path not in self._property_listeners_after:
            self._property_listeners_after[property_path] = []
        self._property_listeners_after[property_path].append(callback)

    def _before_property_change(self, property_path: str, new_value) -> None:
        listeners = self._property_listeners_before.get(property_path, [])
        for listener in listeners:
            listener(property_path, new_value)

    def _after_property_change(self, property_path: str, old_value, new_value) -> None:
        listeners = self._property_listeners_after.get(property_path, [])
        for listener in listeners:
            listener(property_path, old_value, new_value)

# --- Flow Control Helpers ---
class FlowJumpException(Exception):
    """用于模拟 Claw flow-to 跳转的异常"""
    def __init__(self, target_label: str):
        self.target_label = target_label
        super().__init__(f"Flow jump to {target_label}")

def _flow_jump(target_label: str) -> None:
    """触发业务逻辑流转跳转（不记录堆栈）"""
    raise FlowJumpException(target_label)

def _flow_dispatch(handlers: Dict[str, Callable], entry_label: str = "start") -> Any:
    """业务流转调度器"""
    current_label = entry_label
    while current_label in handlers:
        try:
            result = handlers[current_label]()
            return result
        except FlowJumpException as fj:
            current_label = fj.target_label
    raise RuntimeError(f"Unknown flow target: {current_label}")

# ... 其他运行时辅助 ...

class MyClass:
    pass
```

## 实现细节

### 代码生成流程

1. **文件头生成** - 生成 shebang、注释和导入语句
2. **运行时辅助代码** - 生成所有运行时辅助类和方法
3. **元数据生成** - 生成模块级 docstring
4. **顶层块遍历** - 递归遍历所有顶层 IR 块
5. **指令分派** - 根据操作码类型分派到对应的生成方法
6. **__main__ 入口** - 生成程序入口点

### 缩进管理

- 使用 `indentLevel` 变量跟踪当前缩进级别
- 每级缩进使用 4 个空格
- `SCOPE_ENTER` 增加缩进，`SCOPE_EXIT` 减少缩进
- `FUNC_DEF` 增加缩进表示函数体开始
- `FUNC_END` 减少缩进表示函数体结束

### 注释处理

- 支持行内注释（`//` → `#`）
- 支持多行注释
- 支持在行尾追加注释

## 使用方式

```java
PythonCodeGenerator generator = new PythonCodeGenerator();

// 创建 IR
IRGenerator.IRProgram program = new IRGenerator.IRProgram("test.claw");
IRGenerator.IRBasicBlock block = program.createTopLevelBlock();

// 添加指令
IRGenerator.IRInstruction funcDef = block.createInstruction(IRGenerator.OpCode.FUNC_DEF, "myFunction");
IRGenerator.IRInstruction funcEnd = block.createInstruction(IRGenerator.OpCode.FUNC_END);

program.addInstruction(funcDef);
program.addInstruction(funcEnd);

ClawIR clawIR = new ClawIR(program, null, null, null);

// 生成 Python 代码
String result = generator.generate(clawIR);
```
